# 向量資料庫實戰：從理論到企業級應用

> Vector Database in Action: From Theory to Enterprise Applications

---

## 專案概覽

本書是一本全面深入的向量資料庫技術指南，從理論基礎到企業級實戰應用，為讀者提供完整的學習路徑。書中涵蓋向量嵌入、相似性搜尋、索引演算法、FAISS 與 Milvus 等主流工具，並以真實企業場景作為案例驅動教學。

### 目標讀者

- **後端工程師**：希望在現有系統中整合向量搜尋功能
- **資料工程師**：需要建構大規模向量檢索管線
- **機器學習工程師**：將 ML 模型與向量資料庫整合
- **架構師**：設計支援 AI 應用的資料基礎設施
- **技術主管**：評估向量資料庫技術選型

### 先備知識

- Python 程式設計基礎
- 基本的線性代數概念（向量、矩陣）
- SQL 資料庫基本操作經驗
- 了解基本的機器學習概念（非必須但有幫助）
