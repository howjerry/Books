## 三大理論模型 → 書籍章節對應

### 模型一：氣球模型（Balloon Model）

#### 學術概念
- 全靜態負荷（Allostatic Load）
- HPA 軸持續活化
- 耶克斯-多德森定律
- 保存資源理論

#### 對應章節與轉化方式

| 書籍章節 | 學術概念 | Millie 風格轉化 |
|---------|---------|----------------|
| **001：焦慮不是你的敵人** | 適度壓力提升表現 | 「焦慮就像氣球裡的空氣，適量讓你飽滿有彈性」 |
| **121：倦怠不是懶惰** | 全靜態負荷累積 | 「你的氣球不是太小，是被吹太滿了」 |
| **081：界線不是牆** | 關閉進氣閥 | 「說『不』不是自私，是在保護你的氣球不爆掉」 |
| **125：工作邊界** | 能量守恆 | 「下班後繼續想工作，就像氣球的洞一直漏氣」 |
| **187：休息的權利** | 主動洩壓 vs 被動麻痺 | 「滑手機不是休息，那只是暫停進氣，不是排氣」 |

#### 範例篇章：〈你的氣球還剩多少空間？〉

```
「我是心理師，給我 60 秒。」

想像你是一顆氣球。
每一個任務、每一個擔憂、每一個「好的我來」，
都是往你體內吹進一口氣。

氣球不會分辨空氣的來源——
工作壓力是空氣，
興奮期待也是空氣，
幫朋友忙是空氣，
半夜滑手機的焦慮也是空氣。

現在問問自己：
「我的氣球，還剩多少空間？」

如果答案是「快沒了」，
那麼下一個進來的請求，
不管多小，
都可能讓你爆掉。

所以當你說「不」的時候，
你不是在拒絕別人，
你是在保護自己的結構完整。

這不是自私。
這是物理。
```

#### 對應章節與轉化方式

| 書籍章節 | 學術概念 | Millie 風格轉化 |
|---------|---------|----------------|
| **041：住在你腦中的批評者** | DMN 負面循環 | 「那個批評你的聲音，是一條被你走了太多次的路」 |
| **051：你不需要修好自己** | 行為與身份分離 | 「犯錯是一個事件，不是你的本質」 |

| **071：你不是騙子** | 人稱轉換技巧 | 「試著用自己的名字跟自己說話，像對朋友那樣」 |

#### 範例篇章：〈你的大腦是一片森林〉

```
「給我 60 秒。」

你的大腦是一片森林。
每一個想法，都是一條路。

你反覆走的路，會變成大道——
寬敞、平坦、不用思考就能走。

「我不夠好」
「我會搞砸」
「沒有人喜歡我」

如果這些是你每天走的路，
它們就會變成高速公路。
你的大腦會自動把你送到那裡。

但好消息是：
路是可以改的。

每次你刻意選擇另一條路——
「我正在學習」
「我可以試試看」
「這一次不代表每一次」

你就是在森林裡開闢新路徑。

剛開始會很難走，雜草叢生。
但走得夠多，它也會變成大道。

而那條舊的路？
沒人走，就會慢慢長滿草。

這不是正向思考的雞湯，
這是神經科學。
```

### 模型三：茶壺模型（Teapot Model）

|---------|---------|----------------|

| **151：茶壺理論** | 創傷反應正常化 | 「任何人放在那個火上，水都會開」 |
| **034：恐慌發作指南** | 背側迷走神經反應 | 「當水燒乾了，茶壺會自動關機保護自己」 |
| **161：呼吸為什麼有用** | 迷走神經刺激 | 「長吐氣是往茶壺裡加冷水」 |

#### 範例篇章：〈茶壺理論〉（核心篇章）

```
「我是心理師，給我 60 秒。」

今天我想跟你聊聊創傷。

想像你是一個茶壺。
每一次受傷的經歷，
都像是往茶壺底下加火。

如果壺蓋緊緊蓋著——
你壓抑情緒、假裝沒事、告訴自己要堅強——
水溫會不斷升高。

直到有一天，
一件看似微小的事情發生，
茶壺就會沸騰、蓋子炸飛。

「我不懂，我只是打翻了一杯咖啡，
為什麼我會崩潰大哭？」

因為那不只是咖啡。
那是水終於燒開了。

這不代表你壞掉了。
這代表你的身體在說：
「我需要降溫。」

你不需要換一個新茶壺。
你需要的是移開火源，
然後慢慢加入冷水。

這就是為什麼談論感受如此重要。
```

## 轉化原則：從學術到口袋

### 1. 具象化（Concretization）

| 學術術語 | 口袋化比喻 |
|---------|----------|
| 全靜態負荷 | 氣球被吹太滿 |
| 神經可塑性 | 森林裡的小路 |
| 交感神經喚醒 | 水燒開了 |
| 背側迷走神經崩潰 | 茶壺自動關機 |
| HPA 軸 | 身體的警報系統 |
| 杏仁核活化 | 大腦拉響警報 |
| 前額葉皮質 | 大腦的理性經理 |

### 2. 去病理化（De-pathologization）

| 病理化語言 ❌ | 非病理化語言 ✅ |
|--------------|----------------|
| 你有情緒調節障礙 | 你的水燒開了 |
| 你需要被修復 | 你需要降溫 |
| 你的反應過度 | 你的身體在保護你 |
| 你有焦慮症 | 你的氣球被吹太滿了 |
| 你有創傷 | 你經歷了很多熱源 |

### 3. 賦能化（Empowerment）

每篇結尾應該給讀者可以立即嘗試的事情：

- 「試著用自己的名字跟自己說話」
- 「長長地吐一口氣」
- 「問問自己：我的氣球還剩多少空間？」
- 「今天，允許自己打開壺蓋」

## 禁忌清單

### 不要這樣寫 ❌

- 「研究顯示...」（太學術）
- 「你應該...」（太說教）
- 「一切都會好的」（有毒正向）
- 「你有 XXX 症」（病理化）
- 使用專業術語不加解釋

### 要這樣寫 ✅

- 用問題引導
- 用「讓我告訴你一個比喻」
- 用「如果你的朋友這樣，你會怎麼說？」
- 承認困難：「這很難，而且...」
- 正常化：「這是正常的反應」
