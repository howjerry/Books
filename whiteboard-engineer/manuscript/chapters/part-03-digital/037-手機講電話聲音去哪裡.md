# 037 手機講電話時聲音去了哪裡？

> **適合年齡**: 5-9 歲
> **所需時間**: 10-15 分鐘
> **白板需求**: 小型可擦寫白板、黑色與紅色白板筆

---

## 今日主題

你對著手機說話，聲音怎麼跑到遠方奶奶的手機裡？你們中間沒有線連著啊！

---

## 準備工作

### 白板初始圖形
畫兩支手機，中間畫問號表示「聲音怎麼傳過去的？」

### 所需道具（可選）
- 兩個紙杯電話（示範聲音的傳遞概念）
- 手機（實際展示基地台圖示）

---

## 引導問題

用這些問題開啟對話：

1. 「你跟奶奶講電話的時候，聲音怎麼跑過去的？」
2. 「手機沒有線，聲音走哪條路？」
3. 「你有沒有看過那些高高的天線塔？」

---

## 白板圖解步驟

### Step 1：畫出聲音變成數字

畫嘴巴說話，聲波進入手機，變成 0 和 1。

**說這些話**：
「當你對著手機說話，手機會『聽』到你的聲音。然後它會做一件神奇的事——把聲音變成數字！變成好多好多的 0 和 1。電腦和手機都只聽得懂 0 和 1，這樣才能傳送出去。」

### Step 2：畫出無線電波飛向基地台

畫手機發出波浪，波浪飛向一座高高的天線塔。

**說這些話**：
「這些數字會變成『無線電波』從手機飛出去！電波會飛向最近的『基地台』——就是那些高高的天線塔。你在路上、房子屋頂上有沒有看過？它們的工作就是接收和發送手機信號！」

### Step 3：畫出基地台到基地台再到對方

畫信號從基地台經過網路，傳到另一個基地台，再傳到奶奶的手機。

**說這些話**：
「基地台收到你的信號後，會通過網路線把資料傳到很遠的地方。可能經過好幾個基地台和交換中心。最後傳到離奶奶最近的基地台，那個基地台再用無線電波把信號發到奶奶的手機！」

### Step 4：畫出數字變回聲音

畫 0 和 1 進入奶奶的手機，從喇叭變回聲音波。

**說這些話**：
「奶奶的手機收到這些 0 和 1，會把它變回聲音！喇叭就會播出你說的話。整個過程超級快，不到一秒鐘，奶奶就聽到你的聲音了！就像變魔術一樣。」

---

## 核心原理

**一句話版本**：
你的聲音被手機變成數字信號，通過無線電波飛到基地台，經過網路傳到遠方的基地台，再飛到對方手機變回聲音。

**延伸解釋**（供家長參考）：
行動通話的技術流程：
1. **A/D 轉換**：麥克風把聲波轉成類比電信號，再轉成數位信號
2. **編碼壓縮**：用語音編碼器壓縮資料量
3. **調變**：把數位信號轉成無線電波
4. **傳輸路徑**：手機 → 基地台 → 核心網路 → 基地台 → 手機
5. **D/A 轉換**：數位信號還原成聲音

4G/5G 網路使用的頻率：700 MHz - 3.5 GHz（5G 還有更高頻段）

延遲（Latency）：一般約 30-50 毫秒，感覺不到延遲。

## 進階探索（給好奇寶寶）

如果孩子想知道更多：
- **「5G 是什麼？」** → 5G 是最新的手機網路技術！它比 4G 快非常多，而且延遲很低。5G 需要更多基地台，因為它用的無線電波穿透力比較弱，但速度超級快，可以看超高畫質影片完全不會卡！
- **「以前沒有手機怎麼打電話？」** → 以前都用有線的電話！電話線連著電話，一直連到電話公司的交換機，再連到對方家裡。手機的發明讓我們可以隨時隨地打電話，真的很方便！

## 常見問題

**Q: 為什麼有時候聲音會斷斷續續？**
A: 可能是因為無線電波被擋住或太弱了！在電梯裡、地下室、或離基地台很遠的地方，信號會變差。手機收到的資料不完整，聲音就會斷斷續續或聽起來怪怪的。

**Q: 打電話要錢，錢給誰了？**
A: 錢付給電信公司！他們要建造和維護那些基地台、網路設備，還要付員工薪水。你付的電話費就是在使用他們建好的「電話高速公路」！

## 家長小抄

記住手機通話的路徑：
1. 你的聲音 → 變成數字（0和1）
2. 數字 → 無線電波 → 基地台
3. 基地台 → 網路 → 對方附近的基地台
4. 基地台 → 無線電波 → 對方手機
5. 數字 → 變回聲音

關鍵設備：
- 手機（發送/接收）
- 基地台（中繼站）
- 核心網路（交換中心）

核心比喻：
- 基地台 = 郵局的轉運站
- 無線電波 = 看不見的快遞
- 數位信號 = 聲音的「密碼版本」

有趣數字：一通電話的延遲只有 0.03 秒左右！
