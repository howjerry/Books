# 039 為什麼電腦只認識 0 和 1？

> **適合年齡**: 6-9 歲
> **所需時間**: 10-15 分鐘
> **白板需求**: 小型可擦寫白板、黑色白板筆

---

## 今日主題

電腦可以玩遊戲、看影片、寫報告，這麼聰明，但它只認識兩個數字：0 和 1——為什麼？這怎麼可能夠用？

---

## 準備工作

### 白板初始圖形
寫上大大的 0 和 1，旁邊畫一台電腦。

### 所需道具（可選）
- 電燈開關（示範開/關的概念）
- 幾張紙（用 0 和 1 組合成數字）

---

## 引導問題

用這些問題開啟對話：

1. 「你會數數對不對？1、2、3、4、5... 電腦會嗎？」
2. 「電腦只認識 0 和 1，那它怎麼表示 5？怎麼表示 100？」
3. 「燈的開關有幾種狀態？（開和關！）」

---

## 白板圖解步驟

### Step 1：畫出開和關

畫一個電燈開關，標示「開 = 1」「關 = 0」。

**說這些話**：
「電腦裡面有成千上萬個小小的開關。每個開關只有兩種狀態：開或關。我們用 1 代表開，0 代表關。這個簡單的開關就是電腦思考的基礎！」

### Step 2：畫出用多個開關表示數字

畫四個開關排成一排，用 0 和 1 組合出數字。

**說這些話**：
「一個開關只能表示 0 或 1，那怎麼表示大一點的數字呢？用很多開關組合！比如：
- 0001 = 1
- 0010 = 2
- 0011 = 3
- 0100 = 4
- 0101 = 5

看！用 4 個開關，就可以表示 0 到 15 這 16 個數字了！」

### Step 3：畫出字母和圖片也是 0 和 1

畫字母 A，旁邊寫 01000001（A 的 ASCII 碼）。

**說這些話**：
「不只數字，連字母、圖片、音樂都可以用 0 和 1 表示！電腦有一本『密碼本』，規定每個字母對應一串 0 和 1。比如字母 A 是 01000001，B 是 01000010。照片則是把每一個小點的顏色用 0 和 1 記下來！」

### Step 4：畫出為什麼用 0 和 1

畫電路圖的簡化版：有電流 = 1，沒電流 = 0。

**說這些話**：
「為什麼電腦只用 0 和 1，不用更多數字呢？因為最簡單！電線要嘛有電，要嘛沒電，就兩種狀態。如果要分辨 0、1、2、3... 電路就會變得很複雜，而且容易出錯。用最簡單的 0 和 1，電腦反而可以又快又準！」

---

## 核心原理

**一句話版本**：
電腦用 0 和 1（二進位）是因為電路只有「有電」和「沒電」兩種狀態，雖然簡單，但組合起來可以表示任何數字、文字、圖片和聲音。

**延伸解釋**（供家長參考）：
二進位（Binary）的基本概念：
1. **位元（Bit）**：最小單位，0 或 1
2. **位元組（Byte）**：8 個位元，可表示 256 種狀態
3. **字元編碼**：
   - ASCII：用 7-8 位元表示英文字母
   - Unicode：用 16-32 位元表示全世界文字

為什麼用二進位：
- 電路設計簡單（高電壓/低電壓）
- 抗干擾能力強
- 邏輯運算簡潔（AND, OR, NOT）

現代電腦：
- 1 KB = 1024 Bytes = 8192 bits
- 1 張照片可能有幾百萬個 0 和 1

## 進階探索（給好奇寶寶）

如果孩子想知道更多：
- **「一首歌要多少 0 和 1？」** → 一首 3 分鐘的歌大約要 3000 萬個 0 和 1！如果是高品質的音樂，可能要更多。每一秒鐘的聲音都被切成幾萬個小片段，每個片段用一串 0 和 1 記錄。
- **「電腦一秒可以處理多少 0 和 1？」** → 現代電腦超級快！一秒鐘可以處理幾十億甚至幾百億個 0 和 1！這就是為什麼電腦看起來這麼聰明——它做事情快得你根本感覺不到。

---

## 常見問題

**Q: 電腦會數學為什麼這麼聰明？**
A: 電腦其實只會做很簡單的運算！加減乘除、比大小，就這樣。但它做得超級快，一秒鐘可以做幾十億次。很多複雜的事情，都是用這些簡單運算組合起來的！

**Q: 為什麼叫「二進位」？**
A: 因為只用 2 個數字（0 和 1）！我們平常用的是「十進位」，用 10 個數字（0-9）。如果只用 2 個數字，逢 2 就進 1 位，所以叫二進位。比如：1+1=10（二進位的 10 等於十進位的 2）。

## 家長小抄

記住二進位的基本概念：
- **只用 0 和 1**：對應電路的開和關
- **組合表示**：多個位元組合表示更大的數
- **萬物皆可編碼**：數字、文字、圖片、聲音

基本單位：
- 1 bit（位元）= 0 或 1
- 1 byte（位元組）= 8 bits = 256 種可能

二進位轉十進位：
- 1 = 1
- 10 = 2
- 11 = 3
- 100 = 4
- 1000 = 8

核心比喻：
- 電腦的 0 和 1 = 電燈的關和開
- 很多 0 和 1 組合 = 用摩斯密碼傳訊息
- 二進位運算 = 用積木堆出任何形狀
