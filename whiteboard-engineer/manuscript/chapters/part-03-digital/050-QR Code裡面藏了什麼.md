# 050 QR Code 裡面藏了什麼？

> **適合年齡**: 5-9 歲
> **所需時間**: 10-15 分鐘
> **白板需求**: 小型可擦寫白板、黑色白板筆

---

## 今日主題

到處都可以看到那些黑白方塊的圖案，用手機掃一下就能付款、看網站——QR Code 是怎麼把資訊藏在方塊裡的？

---

## 準備工作

### 白板初始圖形
畫一個簡化的 QR Code（大概的黑白方塊樣子）。

### 所需道具（可選）
- 手機（掃描 QR Code 示範）
- 印出的 QR Code（讓孩子觀察細節）

---

## 引導問題

用這些問題開啟對話：

1. 「你有沒有看過這種黑白方塊的圖案？」
2. 「掃一下就能付款或開網站，裡面藏了什麼？」
3. 「為什麼每個 QR Code 長得都不一樣？」

---

## 白板圖解步驟

### Step 1：畫出 QR Code 的結構

畫 QR Code，標示三個角的大方塊「眼睛」。

**說這些話**：
「QR Code 有三個角落都有大大的方塊，就像三隻『眼睛』。這些眼睛很重要！手機掃描時，會先找這三個眼睛，這樣就知道 QR Code 在哪裡、是正的還是歪的。」

### Step 2：畫出黑白方塊代表 0 和 1

放大一小部分，顯示黑色方塊 = 1，白色方塊 = 0。

**說這些話**：
「還記得電腦只懂 0 和 1 嗎？QR Code 就是用黑白方塊來表示 0 和 1！黑色 = 1，白色 = 0。很多很多的黑白方塊排列在一起，組合成一串數字，這些數字就是藏在裡面的資訊！」

### Step 3：畫出資訊如何存進去

畫文字「www.google.com」→ 變成數字 → 變成黑白方塊。

**說這些話**：
「假如我要把『www.google.com』這個網址存進 QR Code，電腦會先把每個字母變成數字（用我們之前說的那種密碼本），然後把數字變成 0 和 1，最後把 0 和 1 畫成黑白方塊。你掃描的時候，反過來把方塊變回文字！」

### Step 4：畫出容錯設計

畫一個 QR Code 部分被遮住，但還是能讀取。

**說這些話**：
「QR Code 很聰明，即使有一部分被擋住或弄髒，還是能讀出來！這是因為它把同樣的資訊存了好幾遍，還有『糾錯碼』可以修復錯誤。有些 QR Code 甚至可以被遮住 30%，照樣能掃描！這就是為什麼可以在中間放 LOGO。」

---

## 核心原理

**一句話版本**：
QR Code 用黑白方塊代表 0 和 1，把文字、網址等資訊編碼成二進位，手機掃描後再解碼還原成原本的內容。

**延伸解釋**（供家長參考）：
QR Code（Quick Response Code）技術：
1. **結構**：
   - 定位圖案（三個角的大方塊）
   - 時序圖案（定位間的線）
   - 資料區域（黑白方塊編碼）
   - 容錯區域

2. **編碼模式**：
   - 數字（0-9）
   - 英數字
   - 位元組（任意資料）
   - 漢字

3. **容錯等級**：L（7%）、M（15%）、Q（25%）、H（30%）

4. **容量**：
   - 最多可存 3000 個英數字元
   - 或 7000 個數字

發明：1994 年日本 DENSO WAVE 公司

## 進階探索（給好奇寶寶）

如果孩子想知道更多：
- **「QR Code 是誰發明的？」** → 日本人在 1994 年發明的！原本是用來追蹤汽車零件的。後來發現它比條碼好用很多——可以存更多資訊、掃描更快、還能修復錯誤，所以越來越普及！
- **「QR Code 和條碼有什麼不同？」** → 條碼是一維的（橫線），只能存 20 幾個數字。QR Code 是二維的（黑白方塊），可以存幾千個字元！而且條碼要對準才能掃，QR Code 歪歪的也能讀。

---

## 常見問題

**Q: 掃陌生的 QR Code 會不會中毒？**
A: 有可能！QR Code 可能連到惡意網站，或是騙你下載壞程式。所以不要亂掃不知道哪來的 QR Code，掃完也要先看看網址是不是正常的再點進去！

**Q: 每個 QR Code 都不一樣嗎？**
A: 存的內容不同，QR Code 當然不一樣。但同樣的內容，用不同的產生器可能會生成不同的圖案，因為排列方式可以有變化。不過掃出來的結果都一樣！

## 家長小抄

記住 QR Code 的原理：
1. **二進位編碼**：黑 = 1，白 = 0
2. **定位眼睛**：三個角幫助識別位置
3. **容錯機制**：可修復 7-30% 損壞
4. **雙向讀取**：橫豎都能存資訊

QR Code vs 條碼：
- 條碼：一維，20 幾個數字
- QR Code：二維，幾千個字元

核心比喻：
- QR Code = 用黑白方塊寫的密碼信
- 三個眼睛 = 告訴手機「從這裡開始讀」
- 容錯碼 = 備份，弄髒也能讀

安全提醒：
- 不要掃來路不明的 QR Code
- 掃完先看網址正不正常
- 警惕引導下載 App 的 QR Code
