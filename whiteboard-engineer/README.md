## 書籍簡介

### 這本書要解決什麼問題？

你是否曾經遇過這些情況：
- 孩子問「為什麼飛機會飛？」，你卻不知道如何用 5 歲孩子能懂的語言解釋
- 想培養孩子的科學思維，但不知道該從何開始
- 買了很多科普書，孩子卻興趣缺缺

這本書提供 **100 個「食譜式」教學單元**，每個單元都像烹飪食譜一樣，有明確的材料（白板 + 彩色筆）、步驟（圖解流程）和預期成果（孩子的理解程度）。

### 核心教學哲學

1. **零準備 (Zero Prep)**: 假設孩子對主題一無所知，降低家長備課壓力
2. **微學習 (Micro-learning)**: 嚴格限制在 10-15 分鐘內，適應兒童注意力
3. **雙向互動 (Interactive)**: 50/50 的時間分配，鼓勵提問與共創
4. **視覺化 (Visual)**: 使用小型可擦寫白板作為主要媒介

## 使用說明

### 如何使用這本書

1. **選擇主題**: 根據孩子的興趣或日常遇到的問題選擇
2. **準備白板**: 準備一塊小型可擦寫白板和彩色白板筆
3. **跟著步驟**: 按照「白板圖解步驟」一步步畫和說明
4. **互動討論**: 留時間讓孩子提問和畫出自己的想法

### 建議的教學頻率

- **最佳**: 每週 2-3 次，每次一個主題
- **最低**: 每週 1 次
- **時機**: 晚餐後、週末早餐時
