# 白板圖解設計規範

本文件提供《白板上的工程學》全書白板圖解的設計標準與繪製指南。

---

## 一、設計原則

### 核心理念

| 原則 | 說明 |
|------|------|
| **簡潔性** | 使用基本幾何形狀，避免複雜細節 |
| **可複製性** | 家長能用白板筆在 10 秒內複製 |
| **漸進式** | 每個步驟只增加 1-2 個元素 |
| **顏色限制** | 最多使用 3 種顏色 |

### 設計限制

- **繪製時間**：每個步驟 < 30 秒
- **元素數量**：每個步驟 < 5 個元素
- **文字使用**：僅使用關鍵詞標註，避免句子
- **白板尺寸**：假設 30cm × 20cm 小型白板

## 二、視覺元素標準

### 基本形狀庫

```
【基本形狀】
○ 圓形    - 代表：粒子、細胞、零件
□ 方形    - 代表：建築、容器、裝置
△ 三角形  - 代表：屋頂、方向、山
— 直線    - 代表：連接、路徑、邊界
～ 波浪線  - 代表：水、聲音、能量波
→ 箭頭    - 代表：方向、流動、力量

【組合圖形】
🏠 房屋 = □ + △
🚗 汽車 = ○○ + □
✈️ 飛機 = —— + △△
👁️ 眼睛 = ○（大）+ ○（小）
❤️ 心臟 = ♡ 形狀
🌍 地球 = ○ + 內部分層
```

### 顏色規範

| 顏色 | 用途 | 範例 |
|------|------|------|
| **黑色** | 主要線條、輪廓 | 結構、外框 |
| **紅色** | 強調、重點標示 | 重要部位、力的方向 |
| **藍色** | 水、冷、電流 | 水流、電線、冷卻 |
| **黃/橘色** | 熱、能量、光 | 火焰、太陽、電力 |
| **綠色** | 自然、生物 | 植物、環保概念 |

### 標註規範

```
【文字標註格式】
- 使用繁體中文
- 字體大小：相對於圖形的 1/4
- 位置：圖形旁邊，用線連接
- 範例：
    ○ ←── 細胞
    □ ←── 地基
```

## 三、各部分圖解風格

### Part 1：基礎設施
**主色調**：黑 + 紅
**常用符號**：建築結構、管線、箭頭

```
【範例：房屋結構】
Step 1:  ═══════  (地面線)
Step 2:  [▓▓▓▓▓]  (地基)
Step 3:  | |      (柱子)
Step 4:   /\      (屋頂)
```

### Part 2：交通機械
**主色調**：黑 + 藍
**常用符號**：輪子、翅膀、箭頭（力的方向）

```
【範例：飛機升力】
        ↑ 升力
    ══════════
  →  機翼截面  →
    ──────────
        ↓ 重力
```

### Part 3：數位世界
**主色調**：黑 + 藍
**常用符號**：方塊（代表資料）、線條（代表傳輸）

```
【範例：二進位】
開關： [ON] [OFF] [ON] [OFF]
數字：   1     0     1     0
```

### Part 4：人體工廠
**主色調**：黑 + 紅
**常用符號**：器官簡化圖、細胞、血管

```
【範例：眼睛構造】
     光線 →
         (  角膜
     [瞳孔]
         (  水晶體
              ●  視網膜
```

### Part 5：地球宇宙
**主色調**：黑 + 藍 + 黃
**常用符號**：圓形（星球）、箭頭（旋轉/軌道）

```
【範例：地球結構】
    ╭──────────╮
    │  ○ 地核  │
    │ ○○○ 地函 │
    │○○○○ 地殼 │
    ╰──────────╯
```

### Part 6：社會經濟
**主色調**：黑
**常用符號**：人形、箭頭、貨幣符號

```
【範例：交易流程】
人A ──魚──→
          [錢]
人B ──蘋果─→
```

## 五、製作流程建議

### 繪圖工具選擇

| 工具 | 優點 | 適用情境 |
|------|------|---------|
| **手繪掃描** | 最接近真實白板感 | 正式出版 |
| **iPad + Procreate** | 可編輯、支援圖層 | 快速迭代 |
| **Figma / Illustrator** | 向量、可縮放 | 電子版本 |

### 製作步驟

1. **草圖階段**：依照章節內容繪製 4 步驟草圖
2. **審核階段**：確認圖解與文字內容一致
3. **正稿階段**：使用標準顏色和線條粗細
4. **標註階段**：添加必要的文字標籤
5. **整合階段**：與排版整合，確認尺寸
