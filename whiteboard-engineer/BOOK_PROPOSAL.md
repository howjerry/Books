## 二、市場分析

### 目標讀者

**主要受眾**：
- 擁有 5-9 歲兒童的家長
- 具備理工背景，希望培養孩子系統性思維的工程師家長
- 對 STEM 教育有興趣但不知如何開始的父母

**次要受眾**：
- 國小低年級教師
- 安親班/課後輔導老師
- 科學營隊講師

### 讀者痛點

1. **語言障礙**：知道原理，但不知道如何用孩子能懂的語言解釋
2. **時間壓力**：工作繁忙，沒有時間準備教材
3. **話題枯竭**：想跟孩子聊科學，但不知道該從何開始
4. **互動困難**：科普書通常是單向閱讀，缺乏親子互動設計

### 競品分析

| 競品 | 優勢 | 本書差異化 |
|------|------|-----------|
| 《科學實驗王》漫畫系列 | 故事吸引人、圖文並茂 | 本書強調親子互動，不是孩子獨自閱讀 |
| 《小牛頓》雜誌 | 內容豐富、定期出刊 | 本書提供可操作的教學步驟 |

| 傳統百科全書 | 內容詳盡 | 本書專為 15 分鐘碎片時間設計 |

### 市場趨勢

1. **STEM 教育熱潮**：政府與民間持續推動科學教育
2. **親子共學風氣**：雙薪家庭重視「質量時間」

## 三、書籍特色

### 核心教學哲學

| 原則 | 說明 |
|------|------|
| **零準備 (Zero Prep)** | 假設孩子對主題一無所知，家長不需要額外備課 |
| **微學習 (Micro-learning)** | 嚴格限制在 10-15 分鐘內，適應兒童注意力 |
| **雙向互動 (Interactive)** | 50/50 的時間分配，鼓勵提問與共創 |
| **視覺化 (Visual)** | 使用小型可擦寫白板作為主要媒介 |

### 獨特價值主張

1. **食譜式結構**：每個主題都有明確的「材料、步驟、成果」
2. **白板圖解**：提供逐步畫圖指引，即使不會畫畫也能教
3. **對話腳本**：提供「說這些話」的建議，降低教學門檻
4. **年齡調整**：每個主題標註適合年齡，可依孩子程度調整
5. **本土化內容**：包含台灣相關案例（如台北101、台灣地震等）

### 內容亮點

- **100 個精心設計的比喻**：如「地基是房子的大雪靴」「眼睛是精密的照相機」
- **完整的對話腳本**：每個步驟都有「說這些話」建議
- **延伸知識**：供家長深入了解的「延伸解釋」區塊
- **日常連結**：將抽象概念與孩子生活經驗結合

### 單元結構（食譜模板）

每個教學單元包含以下部分：

```
1. 今日主題（一句話描述）
2. 準備工作（白板初始圖形、可選道具）
3. 引導問題（3 個激發興趣的問題）
4. 白板圖解步驟（4 個步驟，含「說這些話」腳本）
5. 核心原理（一句話版本 + 延伸解釋）
6. 互動環節（讓孩子動手 + 討論問題）
7. 日常連結（將概念連結到生活中）
8. 進階探索（給好奇寶寶的延伸話題）
9. 常見問題（孩子可能會問的問題）
10. 家長小抄（快速複習的重點摘要）
```

### 精選單元預覽

#### 範例一：飛機為什麼會飛？（#021）

**今日主題**：為什麼那麼重的飛機可以飛上天？機翼的形狀藏著什麼秘密？

**白板圖解重點**：
- 畫出機翼的橫切面（上面彎、下面平）
- 用箭頭表示空氣流動的速度差
- 示範「白努利原理」：空氣流得快，壓力變小

**核心比喻**：機翼就像一塊滑水板，空氣「托」著飛機往上走

#### 範例二：為什麼電腦只認識 0 和 1？（#039）

**今日主題**：電腦明明這麼聰明，為什麼只認識兩個數字？

**白板圖解重點**：
- 畫出開關（開=1，關=0）
- 示範如何用 0 和 1 組合出數字（二進位）
- 解釋「電」的語言就是「開和關」

**核心比喻**：電腦就像一個只會用手電筒打暗號的人——開、關、開開、關關

#### 範例三：錢是什麼？（#084）

**今日主題**：為什麼這張紙可以買東西？錢的「魔力」是怎麼來的？

**白板圖解重點**：
- 畫出「以物易物」的困難
- 畫出「共同約定」的概念
- 畫出錢的演變（貝殼→金屬→紙幣→數位）

**核心比喻**：錢是大家一起玩的遊戲，只要大家都相信，這張紙就有價值
