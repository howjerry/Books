## 💡 核心概念

### Subagents 的三大優勢
1. **情境隔離**：每個 Subagent 獨立運作，避免情境污染
2. **專業化分工**：主 Agent 規劃，Subagent 執行
3. **平行化執行**：大幅縮短處理時間

### 實際效益數據（基於第 4 章案例）
- 執行時間減少 **82%** (45 分鐘 → 8 分鐘)
- 情境使用減少 **98%** (180K → 2.5K tokens)
- API 成本降低 **65%** ($2.40 → $0.85)
- 錯誤率降低 **84%** (12.8% → 2.1%)
