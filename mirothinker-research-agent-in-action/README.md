# MiroThinker Deep Research Agent in Action

> **書名**: 深度研究代理人實戰：基於 MiroThinker 打造企業級 AI 研究助理
> **副標題**: 從開源專案學習交互式縮放與自主代理人架構

---

## 專案概覽

本書是一本聚焦於**深度研究代理人（Deep Research Agent）**的技術實戰指南，以開源專案 [MiroThinker](https://github.com/MiroMindAI/MiroThinker) 為核心教材，帶領讀者從理論基石到工程實踐，完整掌握如何建構一套具備「自我查證」與「長程推理」能力的 AI 研究助理。

### 核心論點

> **智能的奇點不在於無限制地擴大參數（Internal Parameters），而在於強化代理人向外求證、自我修正的「發現式智能」（Interactive Intelligence）。**

本書將深入探討「交互式縮放（Interactive Scaling）」——即如何透過增加模型與環境的交互深度（如 600 次以上的工具調用）來補足模型參數規模的不足。

## 技術框架

### ReAct 範式（Reasoning + Acting）

```
┌─────────────────────────────────────────────────────────┐
│                    ReAct 循環                           │
├─────────────────────────────────────────────────────────┤
│  Thought → Action → Observation → Thought → ...        │
│                                                         │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐         │
│  │ 推理階段 │───▶│ 行動階段 │───▶│ 觀察階段 │──┐      │
│  └──────────┘    └──────────┘    └──────────┘  │      │
│       ▲                                         │      │
│       └─────────────────────────────────────────┘      │
└─────────────────────────────────────────────────────────┘
```

### 核心技術棧

| 層級 | 技術 | 用途 |
|------|------|------|
| **模型層** | Qwen-2.5 (8B-72B) | 基礎推理能力 |
| **框架層** | LangChain / LlamaIndex | Agent 編排 |
| **工具層** | Playwright, Python Interpreter | 環境交互 |
| **檢索層** | Elasticsearch, Milvus | RAG 實現 |
| **部署層** | vLLM, TensorRT-LLM | 推理優化 |

1. **專案驅動開頭**
